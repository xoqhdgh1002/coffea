coffea.nanoaod.methods¶

Awkward 0.x NanoAOD collection methods

This package provides the object method mixins for the various types of NanoAOD collections, as well as a default mapping for collection names to methods.

Classes¶

METVector()

Implements the usual TVector2 methods, storing minimal materialized arrays in polar coordinates

LorentzVector()

Implements the usual TLorentzVector methods, storing minimal materialized arrays in pt-eta-phi-mass coordinates

Candidate()

Implements the usual TLorentzVector methods but also propogates sum of charges

Electron()

NanoAOD electron object

Muon()

NanoAOD muon object

Photon()

NanoAOD photon object

FsrPhoton()

NanoAOD fsr photon object

Tau()

NanoAOD tau object

Jet()

NanoAOD narrow radius jet object

FatJet()

NanoAOD large radius jet object

GenParticle()

NanoAOD generator-level particle object, including parent and child self-references

GenVisTau()

NanoAOD visible tau object
